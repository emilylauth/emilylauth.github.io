# React Portfolio

Forked repo from Raj Shekhar and refactored for my own portfolio. [Cleanfolio Minimal](https://github.com/rajshekhar26/cleanfolio-minimal).

## Instructions

### Setup

```shell
git clone https://github.com/rajshekhar26/cleanfolio
cd cleanfolio
```

Install and run -

```shell
npm install
npm start
```
